Use of Galaxy Image Tools

  • Pan
    This tool allows to pan the image left, right, up, and down. By default this is the selected tool where appropriate.
  • Lasso Select
    This tool is used to facilitate selection on the image. You can use this tool to select radio components on the galaxy images. Double clicking/tapping on the image will deselect all the drawn components and also reset the number of components zero. By default this is the selected tool where appropriate.
  • Box Zoom
    This tool allows to zoom on a selected area of the image.
  • Wheel Zoom
    This tool allows to use the mouse wheel to zoom in and out on the image.
  • Reset
    This tool allows to reset the image to its original state. Please note that this will not reset selections applied using the Lasso Select tool.

About this plot

  • This plot renders a blue point for every galaxy in our database. The database is based on data from The VLA FIRST Survey (http://sundog.stsci.edu/index.html).
    Red stars represent galaxies that have been labeled using this tool.
